## What is Foresight?

Foresight is a **premium AI-powered startup operating system** designed for founders who want a full-stack co-founder — not just a chatbot.

It doesn't just answer questions. It **plans sprints**, **tracks milestones**, **monitors competitors**, **manages your knowledge base**, **runs autonomous research cycles**, and **generates documents** — all from a single dashboard.

> Built with the **Premium Executive Edition** design system — a light-first, glass-panel aesthetic with 0.5px hairline borders, emerald accents, and Geist typography.

## Getting Started

### Prerequisites

| Requirement | Version | Notes |
|-------------|---------|-------|
| **Node.js** | 20.9+ | LTS recommended |
| **PostgreSQL** | 14+ | Or use `npx prisma dev` for zero-config local DB |
| **Ollama** | Latest | Optional — for free local AI, no API key needed |

### Quick Start

```bash
# 1. Clone
git clone https://github.com/sagarmakani01-arch/FORESIGHT-AI-COFOUNDER.git
cd FORESIGHT-AI-COFOUNDER

# 2. Install
npm install

# 3. Configure
cp .env.example .env
# Edit .env with your DATABASE_URL, NEXTAUTH_SECRET, etc.

# 4. Push schema to database
npx prisma db push

# 5. Start development server
npm run dev
```

Open **http://localhost:3000** and sign in.

> **Tip:** Use `npx prisma dev` instead of a manual PostgreSQL install for a zero-config local database that starts automatically.

### Full Environment Variables

```env
# ──────────────────────────────────────────────
# DATABASE
# ──────────────────────────────────────────────
DATABASE_URL="postgres://postgres:postgres@localhost:5432/foresight?sslmode=disable&connection_limit=10&connect_timeout=0&max_idle_connection_lifetime=0&pool_timeout=0&socket_timeout=0"

# ──────────────────────────────────────────────
# AUTHENTICATION
# ──────────────────────────────────────────────
NEXTAUTH_URL="http://localhost:3000"
NEXTAUTH_SECRET="change-this-to-a-random-secret-in-production"

# ──────────────────────────────────────────────
# AI — Ollama (default, no API key needed)
# ──────────────────────────────────────────────
OLLAMA_BASE_URL="http://localhost:11434/v1"
AI_MODEL="llama3.2:latest"

# ──────────────────────────────────────────────
# OPTIONAL: OAuth Providers
# ──────────────────────────────────────────────
# GOOGLE_CLIENT_ID=""
# GOOGLE_CLIENT_SECRET=""
# GITHUB_CLIENT_ID=""
# GITHUB_CLIENT_SECRET=""

# ──────────────────────────────────────────────
# OPTIONAL: CORS
# ──────────────────────────────────────────────
# ALLOWED_ORIGINS="http://localhost:3000"
```

> API keys for OpenAI, Anthropic, and Google Gemini are configured through the **Settings > Connect Providers** UI at runtime, not in `.env`. This keeps keys per-account and out of your codebase.

## Deployment

Foresight is designed for deployment on **Vercel**, **Railway**, **Render**, or any Node.js hosting platform.

### Vercel (Recommended)

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel

# Deploy to production
vercel --prod
```

### Docker

```bash
docker compose -f docker-compose.prod.yml up -d
```

### Manual (Self-Host)

```bash
# Build
npm run build

# Start
npm start
```

### Environment Variables for Production

| Variable | Required | Description |
|----------|----------|-------------|
| `DATABASE_URL` | ✅ | PostgreSQL connection string |
| `NEXTAUTH_URL` | ✅ | Your production domain (e.g., `https://foresight.app`) |
| `NEXTAUTH_SECRET` | ✅ | Random secret (generate with `openssl rand -base64 32`) |
| `OLLAMA_BASE_URL` | Optional | Ollama server URL (if using local AI) |
| `AI_MODEL` | Optional | Default AI model name |
